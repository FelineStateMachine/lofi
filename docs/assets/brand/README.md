# lofi brand assets

The visual system treats **lofi** as both a name and a quiet abbreviation for **lo**cal **fi**rst.
The two halves remain one uninterrupted word and are distinguished by color rather than spacing or
punctuation.

## Assets

| File                          | Use                                                                            |
| ----------------------------- | ------------------------------------------------------------------------------ |
| `lofi-mark.svg`               | Monochrome leaf mark. It inherits `currentColor`.                              |
| `lofi-mark-color.svg`         | Fixed acid-lime leaf mark on a transparent background.                         |
| `lofi-titlemark-on-dark.svg`  | Lime `lo` and cream `fi` for deep-green or other dark backgrounds.             |
| `lofi-titlemark-on-light.svg` | Forest `lo` and jade `fi` for cream, white, or other light backgrounds.        |
| `lofi-hero-art.png`           | Text-free pocket-greenhouse illustration for README and documentation layouts. |
| `lofi-github-banner.png`      | Wide repository/social banner with the complete identity and product metaphor. |

## Palette

| Role                  | Color     |
| --------------------- | --------- |
| Deep forest           | `#18231f` |
| Acid lime             | `#d5f26a` |
| Soft cream            | `#f3faf6` |
| Muted mint            | `#99e5bd` |
| Secondary forest      | `#22332c` |
| Light-background jade | `#39725c` |

The leaf is intentionally low-technology and organic. The self-contained greenhouse communicates
that an app remains useful locally; the smaller connected device makes sync visible but secondary.
Risograph grain and restrained print registration provide the music-genre reference without adding
literal headphones, records, or music notes.

## Raster generation direction

The raster assets use the approved solid leaf and heavy rounded wordmark, a sparse technical-grid
composition, and warm two-ink risograph texture. The hero omits all lettering. The banner renders
the exact lowercase word `lofi` once, with `lo` in acid lime and `fi` in soft cream.
